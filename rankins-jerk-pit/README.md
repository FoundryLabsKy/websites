# Rankin's Jerk Pit — website

Static website for Rankin's Jerk Pit, a charcoal-drum jerk restaurant on Shamrock Road,
Bodden Town, Grand Cayman.

- **Stack:** hand-written HTML + CSS (no build step, no JS framework). Google Fonts
  (Alfa Slab One + Archivo). Inline SVG illustration.
- **Pages:** `index.html` (home: story, menu board, reviews, hours/visit),
  `menu.html` (full menu). `sitemap.xml` for search engines.
- **Run locally:** any static server, e.g. `python3 -m http.server` in this directory,
  then open `http://localhost:8000/`.
- **Live:** https://foundrylabsky.github.io/websites/rankins-jerk-pit/
- **Docs:** `docs/research.md` (Phase 1 research, sourced) and `docs/profile.md`
  (company profile, brand decisions, SEO fix list for the owner).
